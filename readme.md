# SALT Assignment

## Features

1. **Get News List**
    - Fetch news data from API
    - Display news data in list

2. **Detail News (WebView)**
    - Open news detail in WebView

## Tech Stack

The following technologies and tools were used in the development of this application:

- **Jetpack Compose** 
- **Dagger Hilt**
- **Retrofit**
- **Clean Architecture**
